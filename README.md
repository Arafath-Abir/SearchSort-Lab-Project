
# Search & Sort Suite (C, Modular)

A clean, modular C project showcasing **searching** and **sorting** algorithms.

## Algorithms
- Searching: Linear Search, Binary Search
- Sorting: Bubble, Quick, Insertion, Merge, Selection

## Build & Run
```bash
make
./search_sort_demo
```
